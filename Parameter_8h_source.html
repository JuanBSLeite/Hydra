42 /**
43  * @ingroup fit, generic
44  * @brief This class represents named parameters that hold information of value, error, limits and implements the interface with ROOT::Minuit2.
45  *
46  * hydra::Parameter instances are constructible, assignable and copiable in all backends, and because that the storage to hold the name
47  * needs be managed by the user (ex. no std::string support in CUDA).
48  * hydra::Parameter overloads the GReal_t operator()() and the arithmetic operators.
49  * hydra::Parameter instances can be constructed using named parameter semantic or parameter list semantic:
50  @code{cpp}
51  //using named parameter idiom
52  Parameter mean = Parameter::Create()
53  .Name(name)
54  .Value(3.0)
55  .Error(0.000)
56  .Limits(1.0, 4.0)
57  ;
58 
59  //using unnamed parameter idiom
60  Parameter mean(name, 3.0, 0.000, 1.0, 4.0);
61 
62  @endcode
63  *
64  *
65  */
