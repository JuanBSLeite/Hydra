65 
66 ###Introduction###
67 
68 In numerical mathematics, the Gauss–Kronrod quadrature formula is a method for numerical integration
69 (calculating approximate values of integrals). Gauss–Kronrod quadrature is a variant of Gaussian quadrature,
70 in which the evaluation points are chosen so that an accurate approximation can be computed by re-using the
71 information produced by the computation of a less accurate approximation. It is an example of what is called a
72 nested quadrature rule: for the same set of function evaluation points, it has two quadrature rules, one higher order
73 and one lower order (the latter called an embedded rule). The difference between these two approximations is used to estimate
74 the calculational error of the integration.
75 These formulas are named after Alexander Kronrod, who invented them in the 1960s, and Carl Friedrich Gauss.
76 
77 ###Description###
78 
79 The problem in numerical integration is to approximate definite integrals of the form
80 
81 \f[ I = \int_a^b f(x)\,dx \f]
82 
83 Such integrals can be approximated, for example, by \f$n\f$-point [Gaussian quadrature](https://en.wikipedia.org/wiki/Gaussian_quadrature)
84 
85 \f[ \int_a^b f(x)\,dx \approx \sum_{i=1}^n w_i f(x_i), \f]
86 
87 where \f$w_i\f$, \f$x_i\f$ are the weights and points at which to evaluate the function \f$f(x)\f$.
88 
89 If the interval \f$[a,b]\f$ is subdivided, the Gauss evaluation points of the new subintervals never coincide with the previous
90 evaluation points (except at the midpoint for odd numbers of evaluation points), and thus the integrand must be evaluated at every point.
91 Gauss–Kronrod formulas are extensions of the Gauss quadrature formulas generated by adding \f$n+1\f$ points to an \f$n\f$-point
92 rule in such a way that the resulting rule is of order \f$2n+1\f$
93 [[Math. Comp. 66 (1997), 1133-1145] ](http://www.ams.org/journals/mcom/1997-66-219/S0025-5718-97-00861-2/home.html)
94 the corresponding Gauss rule is of order \f$2n-1\f$. These extra points are the zeros of [Stieltjes polynomials](https://en.wikipedia.org/wiki/Stieltjes_polynomials).
95 This allows for computing higher-order estimates while reusing the function values of a lower-order estimate.
96 The difference between a Gauss quadrature rule and its Kronrod extension are often used as an estimate of the approximation error.
97 
98  */

360  void InitNodes()
361  {
362  GReal_t delta = (fXUpper - fXLower)/NBIN;
363  fNodesTable.resize(NBIN);
364 
365  for(size_t i=0; i<NBIN; i++ )
366  {
367  auto node = this->fNodesTable[i];
368  HYDRA_EXTERNAL_NS::thrust::get<0>(node) = 1;
369  HYDRA_EXTERNAL_NS::thrust::get<1>(node) = i;
370  HYDRA_EXTERNAL_NS::thrust::get<2>(node) = this->fXLower + i*delta;
371  HYDRA_EXTERNAL_NS::thrust::get<3>(node) = this->fXLower + (i+1)*delta;
372  HYDRA_EXTERNAL_NS::thrust::get<4>(node) = 0.0;
373  HYDRA_EXTERNAL_NS::thrust::get<5>(node) = 0.0;
374  }
