template<size_t N, hydra::detail::Backend BACKEND>
template<typename FUNCTOR >
void hydra::Decays< N, hydra::detail::BackendPolicy< BACKEND > >::Reweight ( FUNCTOR const &  functor)

Recalculates the events weights according with ; The new weights are the \( w_{i}^{new} = w_{i}^{old} \times functor(Vector4R*...)\).

  • The functor needs derive from hydra::BaseFunctor or be a lambda wrapped using hydra::wrap_lambda function. The functor signature needs to provide the method Evaluate(size_t n, hydra::Vector4R*), for example:
    ...
    struct BreitWigner: public hydra::BaseFunctor<BreitWigner, double, 0>
    ...
    double Evaluate(size_t n, hydra::Vector4R* particles)
    {
    Vector4R p1 = particles[0];
    Vector4R p2 = particles[1];
    return breit_wigner(p1,p2);
    }
    ...
    };
    The same is is valid for a lambda function:
...
double mass = ...;
double width = ...;
auto bw = [ ]__hydra_host__ __hydra_device__(size_t n, hydra::Vector4R* particles )
{
auto p0 = particles[0] ;
auto p1 = particles[1] ;
auto p2 = particles[2] ;
auto m = (p1+p2).mass();
double denominator = (m12-0.895)*(m12-0.895) + (0.055*0.055)/4.0;
return ((0.055*0.055)/4.0)/denominator;
};
*auto breit_wigner = hydra::wrap_lambda(bw);

Obs: the functor need be positive evaluated for all events in the phase-space.

Get a range pointing to a set of unweighted events.

This method will re-order the container to group together accepted events and return a pair of iterators with ready-only access to container. This version takes a functor as argument and will produce a range of unweighted events distributed accordingly. This method does not change the size, the stored events or its weights. The functor needs derive from hydra::BaseFunctor or be a lambda wrapped using hydra::wrap_lambda function. The functor signature needs to provide the method Evaluate(size_t n, hydra::Vector4R*), for example:

...
struct BreitWigner: public hydra::BaseFunctor<BreitWigner, double, 0>
...
double Evaluate(size_t n, hydra::Vector4R* particles)
{
Vector4R p1 = particles[0];
Vector4R p2 = particles[1];
return breit_wigner(p1,p2);
}
...
};

The same is is valid for a lambda function:

...
double mass = ...;
double width = ...;
auto bw = [ ]__hydra_host__ __hydra_device__(size_t n, hydra::Vector4R* particles )
{
auto p0 = particles[0] ;
auto p1 = particles[1] ;
auto p2 = particles[2] ;
auto m = (p1+p2).mass();
double denominator = (m12-0.895)*(m12-0.895) + (0.055*0.055)/4.0;
return ((0.055*0.055)/4.0)/denominator;
};
*auto breit_wigner = hydra::wrap_lambda(bw);

Obs: the functor need be positive evaluated for all events in the phase-space.
